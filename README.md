Premier Coaching Institute Website

A modern and fully responsive Coaching Website built using HTML, CSS, Bootstrap 5, and JavaScript.

Features

- Attractive Home Page
- About Us Section
- Courses Page
- Faculty Details
- Student Results
- Testimonials
- Contact Form
- WhatsApp Support Button
- Google Maps Integration
- Fully Responsive Design
- Smooth Animations and Interactive UI

Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Bootstrap Icons
- AOS Animation Library

Pages Included

- Home (index.html)
- About (about.html)
- Courses (courses.html)
- Contact (contact.html)

How to Run

1. Download or clone the project
2. Open the project folder
3. Run index.html in your browser
